## 데이터 분석 과제 모음

### 주요 과제 목록
- 한국복지패널 데이터 분석
- 국내 방문객 데이터 분석
- 메가커피 스크래핑
- 대한민국행복지도 데이터 분석
- 미국 대학생 정신 건강 데이터 분석
- MYSQL sakila 데이터 분석
- 국내항공운항 데이터 분석
- 미국 항공운항 데이터 분석
- MySQL 분석 코드 파이썬 연동
- 자연어처리
- BC card 실제 데이터 소비 분석
- 차별화된 고객 경험 제공을 위한 혁신방안 기획
